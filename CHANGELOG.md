## v0.2 - 模块化引擎架构拆分 (2026-05-31)

### 🏗️ 架构拆分

- 单 HTML 文件拆分为 **模块化引擎驱动架构**
- 引擎核心层（`engine/`）：样式、数据库、工具函数、模块注册
- 功能模块层（`modules/`）：商店、直播、B站各自独立目录
- 主应用逻辑（`app.js`）：聊天、设置、世界书等非模块功能
- 新增模块只需：创建目录 → 编写 CSS/JS → 在 HTML 中注册一行

### 📁 目录结构

```
engine/
├── styles.css    共享样式
├── db.js         数据持久化层
├── core.js       Engine 模块注册/调度
└── ui.js         UI 工具函数

modules/
├── shop/         商店模块
├── live/         直播模块
├── bilibili/     B站模块
```
